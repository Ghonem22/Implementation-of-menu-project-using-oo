### How to run:

1- open your cmd

2- clone this repo:

       git clone https://github.com/Ghonem22/Implementation-of-menu-project-using-oo.git
       
3- change directory to ./Implementation-of-menu-project-using-oo/Create_menu/

4- python menu.py


This will display all choices you can use:

                1: show_notes,
                2: search_notes,
                3: add_note,
                4: modify_note,
                5: quit
